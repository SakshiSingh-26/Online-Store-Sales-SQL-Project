SQL Sales Analysis (E-Commerce Dataset)

This project contains a complete SQL-based sales analysis performed on an E-commerce dataset.
All queries, insights, and results are documented inside the Word report.

 About the Project
This analysis answers real business questions using SQL, including:
Which products generate the most revenue?
How does each category perform?
Which customers buy the most?
What are the monthly sales trends?
Which SKUs cross specific revenue thresholds?

The project demonstrates:
SQL querying
Grouping & filtering
Joining tables
Extracting insights from raw data
Converting SQL output into business insights
